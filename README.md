# Discord-Foxhole-Logi-Front-Command
Système de gestion de commandes logistiques pour Discord via Custom Command (Slash Command, Modal et Boutons).

## 🚀 Fonctionnement
1. **Commande Slash :** L'utilisateur tape `/logi`.
2. **Formulaire (Modal) :** Une fenêtre s'ouvre demandant la "Position" et le contenu de la "Commande".
3. **Affichage :** Les détails sont envoyés dans le salon nommé `Logi Front`.
4. **Validation :** Un bouton "Livré" permet de supprimer le message une fois la commande traitée.

## 🛠️ Installation
Pour utiliser ces fichiers dans votre bot de commandes personnalisées :
1. Importez le fichier `Slash Command-VeV3M.json`.
2. Importez le fichier `Envoi Commande-i2A3f.json` (assurez-vous d'avoir un salon nommé `Logi Front`).
3. Importez le fichier `Bouton Livré-Dqgrr.json`.

# 📦 Discord Logi-Front System (English)

This project features a custom logistics ordering system for Discord, utilizing **Slash Commands**, **Modals**, and **Buttons**.

## 🚀 How it works
1. **Slash Command:** User types `/logi`.
2. **Modal Form:** A pop-up appears asking for "Position" and "Order" details.
3. **Display:** The information is sent to a channel named `Logi Front`.
4. **Validation:** A "Livré" (Delivered) button allows staff to delete the message once the order is fulfilled.

## 🛠️ Setup
To use these files in your custom command dashboard:
1. Import `Slash Command-VeV3M.json`.
2. Import `Envoi Commande-i2A3f.json` (ensure you have a channel named `Logi Front`).
3. Import `Bouton Livré-Dqgrr.json`.
